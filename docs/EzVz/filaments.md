# Allowed Filament Brands

At the current point the following brands for the given filament type are allowed for usage in EzVz.

| Filament      | Brands                     | Min Temperature   |
| -----------   | -----------                | --------          |
| ABS           | Polymaker                  |   250°C           | 
| ASA           | Polymaker, Prusament, 3D0  |   270°C           |
